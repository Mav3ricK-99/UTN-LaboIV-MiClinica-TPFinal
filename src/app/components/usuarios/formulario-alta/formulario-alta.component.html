Ingreso de Pacientes

El campo nombre es requerido.
El campo nombre debe contener al menos 6 digitos.
El campo apellido es requerido.
El campo apellido debe contener al menos 6 digitos.
El campo edad es requerido.
El campo edad debe ser al menos 1.
El campo edad debe ser como maximo 99.
El dni es requerido.
El DNI debe ser al menos 4.000.000
El DNI debe tener como maximo 8 digitos
El campo Obra social es requerido.
El campo Obra social debe tener almenos 3 digitos.
El correo electronico es requerido.
El correo electronico debe ser valido.
El correo electronico ya se encuentra registrado.
El campo contraseña es requerido.
El campo contraseña debe tener al menos 6 digitos.
El campo repetir contraseña es requerido.
El campo repetir contraseña debe tener al menos 6 digitos.
